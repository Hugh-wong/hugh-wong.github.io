跨语言学习大纲

Wednesday, July 12, 2017

前言

因为经常会要学习到新的语言，新的框架，不同的语言与不同的框架，均有一些不同的框架、或者语言特色，在对一门语言不熟悉的时候，如何快速上手这一门语言和相当的框架就非常重要了。

先不谈具体的语言，罗列一个大纲

4. 包管理器的使用
5. 包的引入
6. 代码的编译
7. 语言的常用应用领域

常用函数及库

1. 常用的数据结构及其基本操作
2. 字符串处理函数
3. 时间函数
4. IO函数
5. 网络函数
6. 多线程/多进程
7. 面向对象的写法

特定领域的框架罗列

测试、部署待库或者工具

1. 单元测试
2. 数据伪造
3. 持续集成

nodejs && javascript

这门新的语言是我最近学的，因此按上面的大纲整理了一下，首先先来认识一下nodejs是什么：

nodejs = commonJS + io.js，是在基于javascript的V8引擎上，增加了能够执行文件IO、网络IO功能。 传统的Javascript相对于nodejs来说，少了io.js，但是能够处理dom。

nodejs基础

1. 只能异步，并有多种写法（promise, await等）
2. nodejs有自己的包管理器npm/yarn等，安装的包可以放在项目下，可以放在公用目录下
3. es6特性可以直接import或者require，浏览器内需要&lt;&gt;这样引入，顺序有影响
4. webpack/gulp等之类的，将前端的代码编译(less-&gt;css, es6js—&gt;js)合并，组织打包等
5. js是前端应用的多；nodejs也可以做一些爬虫或者后端服务器

常见函数

1. new Array(); new Object();
2. 查看文档：js的字符串处理
3. new Date()，注意这里的日期要少一个月，好像有ISODate也可以将字符串转换成日期对象
4. nodejs需要使用iojs, console
5. js直接使用new XMLRequest(); fetch; nodejs可以直接使用开源的request等库
6. cluster
7. ES6新特性有class与继承，之前需要使用prototype来写类方法

前端开发框架

1. jquery，封装了dom操作； reactjs, angularjs, vue等
1. 官网下载，自己在html里&lt;src&gt;引入
2. 不同的框架写法不一样，有些支持es6将这些进行打包和安装
3. 参考各自的文档页

后端web开发框架

1. web框架: express, egg.js等
2. ORM: sequelize, bookshelf, moogoose等
3. 配置：config
4. 日志: log4js等

部署工具

1. 前端一般只需要静态的服务器，开发阶段也可以使用mockserver等
2. 进程守护等使用pm2

测试工具

mocha, chai等

java （待完成）

java在组织web服务器的代码的时候，通常会按这样的职责划分清晰：

Controller —> 负责接收请求，解析请求数据，调用真正的接口，并返回。 DTO/Plain Class —> DTO是数据转换类(data transfer object)，和基本的get/set IAction —> 负责定义接口类 ActionImpl —> 负责定义接口的实现

maven，算是java里的一个包管理器吧。 类似于npm，包括安装包依赖，打包等。

tomcat，运行war（类似于jar包，但包含一些静态资源）包的web服务器。部署方式是把war包放在对应的目录就行了。
